# Extension Icons

이 폴더에는 확장 프로그램 아이콘이 필요합니다.

## 필요한 아이콘 파일
- `icon16.png` - 16x16 픽셀
- `icon48.png` - 48x48 픽셀
- `icon128.png` - 128x128 픽셀

## 아이콘 생성 방법

### 방법 1: 온라인 변환기 사용
1. `icon.svg` 파일을 https://cloudconvert.com/svg-to-png 에 업로드
2. 각 크기(16, 48, 128)로 변환하여 다운로드

### 방법 2: 임시 아이콘 사용
개발 테스트용으로 아래 Base64 PNG를 사용할 수 있습니다.

## 임시 테스트용 (개발 시)
manifest.json에서 아이콘 설정을 제거하고 테스트할 수 있습니다.
